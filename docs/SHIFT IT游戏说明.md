# SHIFT IT游戏操作/运行说明

###### 团队：队名占位符

#### 1.游戏简介

本游戏属于2D横板平台跳跃通关游戏。鉴于赛题给定的“shift”主题，本游戏核心玩法在于“shift键”，通过赋予其不同的功能以增强游戏性。不同关卡中的shift键功能不同，并紧密围绕“shift”展开，主要有以下几关和功能：

- level_1：冲刺
- level_2：角色体型变大/小
- level_3：和怪物交换位置
- level_4：重力翻转
- level_5：切换场景/玩家可见性
- level_6：切换操作对象（玩家/目标点）
- level_7：切换风向
- level_8：交换虚影/玩家位置（可理解为不定向位移）
- level_9：切换刺有刺无

#### 2.游戏操作说明

本游戏采用最基础的交互按键：“A”向左走，“D”向右走，“SPACE”跳跃，以及核心的“shift”释放该关卡的shift技能。

#### 3.制作说明

本游戏采用godot引擎制作，使用codex和gemini作为编程辅助和项目源码结构整理辅助，具体参见随包附上的“SHIFT IT AI使用报告”。
